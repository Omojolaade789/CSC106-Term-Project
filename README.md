# Student Portfolio and Academic Management Website (CSC 106 Term Project)

A responsive, multi-page portfolio and academic management site built with plain HTML, CSS and JavaScript.

## Pages
- `index.html` - Homepage (photo, welcome message, biography, audio ident)
- `about.html` - Education, career aspirations, skills, hobbies
- `projects.html` - Three sample projects with screenshots and links
- `planner.html` - Interactive academic planner (add / complete / delete tasks)
- `contact.html` - Contact form with JavaScript validation
