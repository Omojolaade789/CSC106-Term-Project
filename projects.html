Projects

Three projects I have built while learning web development. Each one pushed me a little further than the last.

Campus Course Companion

A revision notes website that organises summaries, key definitions and past questions for first-year courses. Built as a multi-page static site, it taught me how to structure semantic HTML and share one stylesheet across many pages.

  • HTML5
  • CSS
  • Responsive design

Naija Recipe Box

A collection of Nigerian recipes displayed in a responsive card grid, complete with cooking times and ingredient lists. This project is where CSS Grid finally clicked for me, and where I learnt to make images behave on small screens.

  • CSS Grid

Quick Quiz

A multiple-choice quiz game that keeps score, gives instant feedback and shuffles its questions on every play. My first real JavaScript project, it introduced me to arrays, functions, event handling and updating the page without reloading it.
